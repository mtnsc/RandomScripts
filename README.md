A collection of scripts I have created for day-to-day use and quick task automations.

### üìä Overview

Here's a list of the scripts included in this repository:

1. **change-author:** Rewrite a git repo's history to replace the author's name and email
1. **docker-clear:** Stop processes, remove exited containers and delete unused Docker images
1. **repo-exec:** Recursively execute a command inside all subfolders (1 level)
1. **repo-search:** Recursively search for files while ignoring useless folders 
1. **samsung-debloat:** Remove useless apps and services pre-installed on Samsung phones
1. **search-product:** List Apigee's products associated with a specific proxy
1. **webm2mp4:** Download and convert WebM videos into MP4

- Download the script by cloning the repo, saving the raw file or copy/pasting it to your computer
